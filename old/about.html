This is a tracker intended for use with the Kingdom Hearts 2 Final Mix Randomizer by Valaxor, for the Garden of Assemblage (GoA) mod by Sonicshadowsilver2.

It allows the tracking of progress such as world completion, drive forms, magic, etc. For a more detailed explanation, see Help.

It was initially developed by Kokemon198, however has since largely been rewritten by Zax with an entirely different codebase.

The GitHub repository with the code for this site can be foundhere.

If you have any questions/issues you can reach out through Discord (you can find me in therando server), Twitter (@Z_ax__), or GitHub.
